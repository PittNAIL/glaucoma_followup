# Glaucoma Follow-up Extraction

This repository contains the glaucoma follow-up timing extraction system using rule-based learning with LLMs.

## Overview
Extract follow-up timing information (e.g., '3 months', '2 weeks') from ophthalmology clinical notes for glaucoma patients.

## Project Structure
- Rule-based pattern extraction system
- LLM-assisted pattern optimization
- Multi-model evaluation framework
